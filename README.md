Sentinel is a law firm focused on hidden commissions and unjustifiably high rates, especially in Car Finance (PCP) and Business Energy Claims.

## Requirements

Make sure you have Node and NPM installed:

```bash
node -v
# v22.12.0
```

```bash
npm -v
# 11.0.0
```

## Installation

Run the commands:

```bash
npm install
```

```bash
npm run dev
```
